# mvp
Minimum Viable Product

## Idea
- 아이디어 탐색: 프롬프트
 - 역할(Role): 너는 항상 최악의 경우를 가정하는 비관적으로 아이디어를 판단할 줄 아는 사람이다.
 - 작업(Task):
   - 아이디어가 가질 수 있는 타겟을 전체시장, 유효시장, 수익시장으로 세분화
   - 아이디어가 가질 수 있는 수익 주가 개형에 대해서 그려줘
   - Background, Problem, Solution, Scale-up, History 평가 지표에 따라 판단해줘
 - 상황(Context)
   - 단기적으로 잠재고객을 확인하고 문제를 분석해야 함
 - 형식(Format)
   - 아이디어 제목
   - 목표 타겟(잠재 고객)과 문제(유저 스토리)
   - 아이디어 주가 개형 -시각적인 그래프로 그려야 함
   - BPSSH 평가(표)
